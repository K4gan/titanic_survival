# Titanic survival prediction

## Overview

Binary classification project predicting passenger survival on the Titanic using scikit-learn. The pipeline covers exploratory analysis (notebook), training two baseline models with preprocessing and comparison (logistic regression vs random forest), and CLI inference loading serialized pipelines.

## Dataset

- **Source:** `seaborn.load_dataset("titanic")` — bundled with Seaborn; no manual download.
- **Target:** `survived` (0 = did not survive, 1 = survived).
- **Features used in training:** `pclass`, `sex`, `age`, `sibsp`, `parch`, `fare`, `embarked`.
- **Notes:** Rows with missing `embarked` are dropped before the train/test split. Numeric columns use median imputation + scaling; categoricals use most-frequent imputation + one-hot encoding (unknown categories ignored at inference).

## How to run

1. Create a virtual environment and install dependencies (example):

   ```bash
   python -m venv .venv
   source .venv/bin/activate   # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```

2. **EDA:** open and run `titanic_eda.ipynb` with Jupyter.

3. **Train:**

   ```bash
   python train.py
   ```

   This writes `models/logistic_regression.joblib`, `models/random_forest.joblib`, and `models/comparison.json`.

4. **Predict (CLI):**

   ```bash
   python predict.py --model rf --pclass 3 --sex male --age 22 --sibsp 0 --parch 0 --fare 7.25 --embarked S --proba
   ```

   Use `--model lr` for logistic regression. `--proba` prints estimated probability of survival.
